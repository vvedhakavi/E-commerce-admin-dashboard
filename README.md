# E-Commerce Admin Dashboard

## Project Description

E-Commerce Admin Dashboard is a modern admin panel built with React and Tailwind CSS. It helps administrators manage products, customers, orders, and sales analytics through an easy-to-use interface.

## Features

* Dashboard Overview
* Product Management
* Customer Management
* Order Management
* Revenue Analytics
* Responsive Design
* Sidebar Navigation
* Search and Filter Functionality
* CRUD Operations
* API Integration using Axios

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Tailwind CSS
* Axios
* Redux Toolkit
* React Icons
* Recharts

## Installation Steps

1. Clone the repository

git clone https://github.com/vvedhakavi/E-commerce-admin-dashboard.git

2. Navigate to project folder

cd E-commerce-admin-dashboard

3. Install dependencies

npm install

4. Start development server

npm run dev
